# todojure

A website written in noir. 

## Usage

    :::bash
    lein deps
    lein run

Note: to generate Noir projects w/ Clojure 1.3 using Leiningen you can use [this](http://clojars.org/rplevy/lein-noir) plugin:

    :::bash
    lein plugin install rplevy/lein-noir 1.3.0
